# Hapi Pusher Chatkit Server 💬
> An auth/token server for Chatkit, written in Hapi!

## Install
```
npm i
```

## Running
```
INSTANCE_LOCATOR=YOUR_INSTANCE_LOCATOR SECRET_KEY=YOUR_CHATKIT_SECRET_KEY npm start
```
